Element, Class and I.D.

What is a CSS selector?

  • Selectors are patterns used to identify, and select differenct elements withing an HTML documen
  • These selectors are used to style the frontend of a website
  • There are over 50 CSS selectors (57 and growing according to W3schools

Element, or type

  • Targets a specific HTML element throughout the whole code
  • Put the element tag name followed by curly brackets

Class Selectors

  • Class selectors are case sensitive
  • Case selectors can be used as many times as needed
  • These are suppored by all modern browsers
  • Select classes by putting a “.” before the class name, followed by the curly braces

ID Selectors

  • Case Sensitive
  • Selected by using the “#” signal and then the id name
  • You can only have one instance of an ID in an html document
